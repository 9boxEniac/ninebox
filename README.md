# 💙 Portal de Estágio - Nine Box Grid

Sistema elegante e profissional de avaliação para professores e estagiários com implementação do Nine Box Grid.

## 🎨 Design Azul Elegante & Profissional

- **Paleta de cores azul**: Tons de azul profissional e elegante
- **Tipografia elegante**: Fonte Poppins para maior sofisticação
- **Gradientes suaves**: Transições delicadas entre tons de azul
- **Animações fluidas**: Movimentos suaves e naturais
- **Sombras delicadas**: Profundidade sutil com tons azuis

## 🚀 Funcionalidades

### 📋 Avaliações Integradas
Três tipos de avaliação em uma única interface:

1. **Professores**: Avaliação com pontuação (1-5 estrelas) nos critérios:
   - Pontualidade
   - Comunicação  
   - Didática
   - Disponibilidade
   - Relacionamento

2. **Estagiários**: Avaliação apenas por comentários nos aspectos:
   - Pontualidade
   - Comunicação
   - Desempenho Técnico
   - Proatividade
   - Trabalho em Equipe

3. **Nine Box Grid**: Avaliação cruzando Performance × Potential

### 📊 Nine Box Grid
Sistema de avaliação cruzando **Performance** (desempenho atual) com **Potential** (potencial de crescimento):

#### Categorias do Nine Box:
- **Superstar** (Alto Performance + Alto Potential) ⭐
- **Estrela** (Médio Performance + Alto Potential) 🌟  
- **Enigma** (Baixo Performance + Alto Potential) 🔮
- **Especialista** (Alto Performance + Médio Potential) 👩‍💼
- **Núcleo** (Médio Performance + Médio Potential) 💎
- **Dilema** (Baixo Performance + Médio Potential) 🤔
- **Âncora** (Alto Performance + Baixo Potential) ⚓
- **Trabalhador** (Médio Performance + Baixo Potential) 👷‍♀️
- **Questão** (Baixo Performance + Baixo Potential) ❓

### 👥 Gestão de Pessoas
- Cadastro de professores e estagiários
- Upload de fotos com preview elegante
- Consulta com filtros intuitivos
- Relatórios visuais completos

## 🎯 Como Usar

### 1. Cadastrar Pessoas
1. Acesse **Cadastrar** no menu principal
2. Escolha o tipo: Professor ou Estagiário
3. Preencha os dados e adicione foto (opcional)
4. Clique em **Cadastrar**

### 2. Fazer Avaliações
1. Acesse **Avaliações** no menu principal
2. Escolha entre três tipos:
   - **Avaliar Professor**: Sistema com estrelas + comentário
   - **Avaliar Estagiário**: Apenas comentários detalhados
   - **Nine Box**: Avaliação Performance × Potential

### 3. Nine Box Grid
1. Selecione "Nine Box" na página de avaliações
2. Escolha Professor ou Estagiário
3. Selecione a pessoa
4. Avalie **Performance** (Baixo/Médio/Alto)
5. Avalie **Potential** (Baixo/Médio/Alto)
6. Adicione comentário (opcional)
7. Salve a avaliação

### 4. Visualizar Histórico
- Histórico unificado com todos os tipos de avaliação
- Badges coloridos para fácil identificação
- Filtros por tipo de pessoa
- Ordenação cronológica
